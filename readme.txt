方法一：把文件夹拷过去（最简单）

  整个项目就是一堆文件，拷贝 E:\Claude\vocab-quest\ 文件夹到另一台电脑，然后在那个文件夹里运行：

  npx serve -l 3456

  浏览器打开 http://localhost:3456 就能玩。

  ▎ 缺点：每台电脑的进度是独立的（数据存在 IndexedDB 里）
